# PlayerManager

## Referencing
```js
import {
    playerManager,
} from './api/BeAPI.js'
```

## getPlayerNames
- Returns an array of players names.
```js
playerManager.getPlayerNames()
```

- Returns an array of player objects.
```js
playerManager.getPlayers()
```
